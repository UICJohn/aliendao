# 大语言模型下载站

本站为HuggingFace（AI资源托管社区）的资源下载镜像网站，为开发者提供模型下载加速服务。

将模型下载到本地文件夹，AutoModel.from_pretrained的pretrained_model_name_or_path参数指向到本地文件夹。

## 下载方法：

### 方法1

从本站搜索模型下载。

### 方法2

按用https://github.com/git-cloner/aliendao方法从huggingface官网断点续传下载。
